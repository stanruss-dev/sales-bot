## ✨ Возможности

| Функция | Описание |
|---|---|
| 🧠 **AI-ответы** | Использует Claude API для умных ответов на основе базы знаний |

| 🔔 **Уведомления** | Оповещает менеджера при горячем лиде |

## 🏗 Архитектура

```
Клиент (Telegram)
       │
       ▼
  aiogram Bot
       │
       ├── /start → Приветствие + начало воронки
       ├── Текст   → AI-обработка через Claude API
       └── Кнопки  → FSM переходы по воронке
                         │
                         ▼
                  Claude claude-sonnet-4-6
                  + База знаний (RAG)
                         │
                         ▼
               Ответ клиенту + следующий шаг
```

**Воронка продаж (FSM):**
```
СТАРТ → ЗНАКОМСТВО → ВЫЯВЛЕНИЕ ПОТРЕБНОСТЕЙ → ПРЕЗЕНТАЦИЯ → РАБОТА С ВОЗРАЖЕНИЯМИ → СДЕЛКА
```

## 🚀 Быстрый старт

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/YOUR_USERNAME/sales-bot.git
cd sales-bot
```

### 2. Установите зависимости

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 3. Настройте переменные окружения

```bash
cp .env.example .env
```

Отредактируйте `.env`:

```env
TELEGRAM_BOT_TOKEN=your_telegram_bot_token
ANTHROPIC_API_KEY=your_anthropic_api_key
MANAGER_CHAT_ID=your_telegram_id  # для уведомлений о горячих лидах
```

Получить токены:
- **Telegram Bot Token** → [@BotFather](https://t.me/BotFather)
- **Anthropic API Key** → [console.anthropic.com](https://console.anthropic.com)

### 4. Заполните базу знаний

Отредактируйте файлы в папке `knowledge_base/` — добавьте информацию о вашей компании, товарах и ценах.

### 5. Запустите бота

```bash
python main.py
```

## 📚 База знаний

База знаний хранится в виде Markdown-файлов в папке `knowledge_base/`. Бот загружает все файлы при старте и использует их как контекст для ответов.

### Структура файлов

**`company.md`** — информация о компании:
```markdown
# О компании
Название, история, миссия, контакты, режим работы...
```

**`products.md`** — каталог товаров:
```markdown
# Каталог товаров
## Товар 1
- Описание, характеристики, преимущества
```

**`pricing.md`** — цены и акции:
```markdown
# Цены и скидки
## Текущие акции
- Скидка 20% на первый заказ...
```

**`objections.md`** — работа с возражениями:
```markdown
# Работа с возражениями
## "Дорого"
Ответ: ...
```

> 💡 **Совет:** Чем подробнее база знаний — тем точнее ответы бота.

## 🛒 Воронка продаж

Бот ведёт клиента через 5 этапов:

| Этап | Триггер | Действие бота |
|------|---------|---------------|
| 1️⃣ **Знакомство** | `/start` | Приветствие, вопрос о цели визита |
| 2️⃣ **Выявление потребностей** | Первый вопрос клиента | Уточняющие вопросы об интересах |
| 3️⃣ **Презентация** | Понятна потребность | Рассказ о подходящем товаре |
| 4️⃣ **Работа с возражениями** | Сомнения клиента | Ответы из базы знаний |
| 5️⃣ **Закрытие** | Готовность купить | Оформление + уведомление менеджера |

## ⚙️ Конфигурация

| Переменная | Описание | По умолчанию |
|---|---|---|
| `TELEGRAM_BOT_TOKEN` | Токен Telegram-бота | обязательно |
| `ANTHROPIC_API_KEY` | Ключ Claude API | обязательно |
| `MANAGER_CHAT_ID` | Telegram ID менеджера | опционально |
| `AI_MODEL` | Модель Claude | `claude-sonnet-4-6` |
| `MAX_HISTORY` | История сообщений | `10` |
| `LOG_LEVEL` | Уровень логов | `INFO` |
